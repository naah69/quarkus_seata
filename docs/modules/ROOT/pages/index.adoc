== Installation

If you want to use this extension, you need to add the `io.quarkiverse.seata:quarkus-seata` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.seata</groupId>
    <artifactId>quarkus-seata</artifactId>
</dependency>
----
